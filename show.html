ie版本控制的相关问题

发布时间：2015-08-18 　 点击数：90 　作者：admin

最近做社区的官网需要在头部嵌入一个swf可以让玩家在当前页面就可以进入游戏，所有就又要研究flash播放的兼容问题，相关的资料网上一大堆，之前看老外博客的时候看到了一个奇葩的方法感觉很实用，拿过来试试：

var _IE = (function(){

    var v = 3, div = document.createElement('div'), all = div.getElementsByTagName('i');

    while (

        div.innerHTML = '<!--[if gt IE ' + (++v) + ']><i></i><![endif]-->',

        all[0]

    );

    return v > 4 ? v : false ;

    

}());

当浏览器处于9或者以下版本时就会返回相应数字，否则返回FALSE

这个方法利用if gt IE得到ie版本

经过亲自测试，返回结果很稳定。
